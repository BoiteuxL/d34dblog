My personal configuration files for GNU/Linux, specifically tailored for the Gnome desktop environment. These files include settings and customizations that enhance my daily workflow and user experience while providing a nice UI.

{% endblock %} {% block features %}

Distro and desktop environment

When it came to choosing a Linux distribution, I always struggled to find the one that suited my needs. I have tried many, such as Debian, Ubuntu, Fedora, Linux Mint, and even Arch Linux. I eventually settled on EndeavourOS, which is based on Arch Linux and provides a user-friendly experience while maintaining the flexibility and power of Arch. I have been using EndeavourOS for a while now, and I am very satisfied with its performance and stability. It has become my go-to distribution for both my laptop and desktop computers.

I use Gnome as my desktop environment. While I have tried other environments like KDE, Cinnamon and even Window Managers such as i3 and Hyprland, I find Gnome to be the most suitable for my needs. It is fluid, smooth, good-looking out of the box and the multiple workspaces feature is a must-have for me.

Theme and icons

I use the Catppuccin Mocha theme with a purple accent, which provides a soothing pastel color palette that is easy on the eyes and beautiful to look at. The theme is complemented by the Tela icon set (Dracula), which offers a clean and modern look that complements the overall aesthetic.

Furthermore, I am a huge Star Wars nerd, so, for my laptop (named Acclamator), I added a nice wallpaper featuring the Acclamator-class assault ship.

And for my desktop (named Venator), I use have wallpaper featuring the Venator-class Star Destroyer.

I also have a couple of configuration files for various apps, such as Git, Starship or Fastfetch.

Extensions

Gnome allow users to customize and enhance the desktop environment by installing addons<. They provide flexibility to tailor the interface to individual preferences, such as adding applets, changing window management, or integrating productivity tools. I have curated a selection of addons that improve my workflow and user experience. Some extensions I use includes:

  • Dash to Dock
  • Just Perfection
  • Forge
  • User Themes
  • Runcat
{% endblock %} {% block technologies %}

This project is just a bunch of Bash scripts which installs packages, downloads themes and icons, imports settings, and installs extensions. It is designed to be run on a fresh installation of EndeavourOS with Gnome.
